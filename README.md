## 🚀 Featured Project

### [⚡ Fake WiFi Detection System](https://github.com/Rudra1004/fake-wifi-detector)
> A real-time Python desktop tool that detects **Evil Twin attacks** and rogue access points

- 🔴 **Evil Twin Detection** — identifies same SSID with multiple BSSIDs (MAC addresses)
- 🟡 **Open Network Warning** — flags unencrypted networks
- 🟡 **Weak WEP Detection** — alerts on crackable legacy encryption
- 🟡 **Rogue AP Signal Analysis** — detects suspiciously strong signals
- Built with Python, Tkinter, netsh — runs on Windows, no extra hardware needed
